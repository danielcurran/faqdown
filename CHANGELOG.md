# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-06-17

### Added
- GameFAQs walkthrough → Markdown converter (standalone CLI, zero dependencies)
- Table of contents with anchor links to every section
- Proper heading levels matching guide structure
- Content-aware block reformatting (prose, tables, ASCII art, stat blocks, decorative, boss cards, shop listings, character sheets, character portraits)
- Paragraph break detection at walkthrough instruction steps (Go, Turn, Take, Enter)
- Decorative header stripping (`// DUNGEON #2` → clean bold text)
- Equipment table detection and markdown pipe table conversion
- Split-guide script for mobile-friendly per-section output (`guide/index.md` + `toc.json` + `meta.json` + one file per section)
- `meta.json` with title, subtitle, author, source, and attributionHtml
- RetroAchievements integration: `fetch-achievements.js`, `validate-achievements.js`, `achievements.json` schema, checklist generation
- RA Comments API integration with `communityTips` field
- Achievement type taxonomy (story, missable, collectible, challenge, secret, progress)
- `achievements.md` generated by `split-guide.js` with missable table + by-section checkboxes
- opencode agent skills: `faqmd` (conversion), `retroachievements` (AI achievement matching), `reformat-review` (edge-case polish)
- `sync-skills.js` to mirror skill files globally
- Walkthrough format detectors: plain-number, bracket-ccode, roman-numeral

### Fixed
- Strip `\r` from raw text for correct `\n{2,}` splits
- Aggressive paragraph break regex no longer fragments prose
- Decorative text detection no longer discards non-decorative prose lines
- ASCII art dungeon headers classified correctly (map type)
- Single-line dungeon entrance markers stripped
- MIXED tags suppressed for art+table/stat block combos
- Format decorative text false positive on stat blocks
- Player tips section 0.1 no incorrectly placed

### Changed
- Modularized reformatter into `lib/reformat/{detect,format,classify,index}`
- Extracted conversion core to `lib/convert-core.js`
- Tests rewritten to import from modules
- `lib/cli.js` with `--help`, input/output validation
- `package.json` with `npm test`, `npm run convert`
- Section titles escaped in headings
- Fetch with timeout and status check
- Dynamic header-end trim (no longer hardcoded line count)
- Stripped title used in `toc.json` to prevent doubled section number prefix
- `isStatBlock` threshold lowered 40% → 30%
- Decorative threshold lowered to 8
- Private content repo separation (walkthroughs in `gamemds` repo)

### Removed
- art-modernize skill (replaced by reformat-review)
- Deploy workflow (moved to gamemds repo)
- All generated walkthrough files from git
- Dead npm dependencies
